# Airbnb Clone - User Stories

## GUEST USER STORIES
### Authentication & Profile

- As a guest, I want to register an account so that I ca book properties and manage my reservations.
- As a guest, I want to login my account so that I can access my booking history and personal information. 
- As a guest, I want to manage my profile information so that hosts can learn more about me and I can keep my details updated.
