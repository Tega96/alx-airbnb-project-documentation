# Airbnb Clone - User Stories

## GUEST USER STORIES
### Authentication & Profile
    
    **AUTH-001** - As a guest, I want to register an account so that I ca book properties and manage my reservations.
    **AUTH-002** As a guest, I want to login my account so that I can access my booking history and personal information. 
    **AUTH-003** As a guest, I want to manage my profile information so that hosts can learn more about me and I can keep my details updated.
